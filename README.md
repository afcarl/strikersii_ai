# strikersii_ai
AI plays strikers 2.


1. Repository clone     
`$ git clone https://github.com/aikorea/strikersii_ai/

2. Submodule init    
`$ git submodule init

3. Mame build or download from http://mamedev.org/release.html    
`$ cd mame
`$ make

4. run randome move script.    
`$ ./mame64 s1945ii -script random_move.lua -window
